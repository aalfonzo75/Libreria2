Acerca de

Proyecto de ejercitación del curso de Programación Web Full Stack de EGG.

Desarrollo de un sistema web de reserva de libros con programación orientada a objetos (POO) en JAVA, modelo de diseño MVC, utilizando una base de datos MySQL, JPA Repository para persistir objetos y Spring Boot como framework de desarrollo web. Administración de Libros, Autores, Editoriales, Clientes y Préstamos (guardar/modificar, listar, dar de baja (CRUD)).

Para realizar un préstamo se registra el libro a prestar y el cliente asociado. Se registra la fecha en la que se efectuó el préstamo y la fecha en la que fue devuelto el libro.
